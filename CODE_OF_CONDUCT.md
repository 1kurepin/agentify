# Code of Conduct

## Our Standard

This project aims to provide a respectful, inclusive, and constructive collaboration environment.

## Expected Behavior

- Be respectful in communication and feedback.
- Focus on ideas and technical outcomes, not personal attacks.
- Assume good intent and ask clarifying questions before escalating.
- Keep discussions relevant, factual, and actionable.

## Unacceptable Behavior

- Harassment, discrimination, or hate speech
- Personal insults, trolling, or sustained disruption
- Sharing private information without consent
- Any conduct that undermines psychological safety in project spaces

## Scope

This Code of Conduct applies to project spaces, including issues, pull requests, discussions, and related communications.

## Enforcement

Maintainers may remove, edit, or reject comments and contributions that violate this Code of Conduct, and may apply temporary or permanent participation restrictions.

## Reporting

To report behavior that violates this Code of Conduct, open an issue and request maintainer follow-up.
