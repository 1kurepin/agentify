# Brand Assets

This folder stores Agentify visual identity assets grouped by usage.

## Structure

- `logos/` - wordmark lockups and logo variants (dark, light, transparent, preview)
- `marks/` - standalone mark assets
- `social/` - social preview images
- `cards/github-repo/` - repository card artwork (SVG source + PNG export)
- `guidelines/` - brand style documentation

## Recommended Files

- README logo (dark): `logos/agentify_logo_dark_tight.png`
- README logo (light): `logos/agentify_logo_light_tight.png`
- Mark (SVG): `marks/agentify_mark.svg`
- GitHub social preview: `social/agentify_github_social_preview_1280x640.png`
- Repo card source: `cards/github-repo/agentify-github-repo-card.svg`
