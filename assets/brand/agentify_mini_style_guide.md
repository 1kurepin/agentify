# Agentify mini style guide

## Brand tone
- Engineering‑first, reliable, modern, structured
- Minimalism: **no visual noise**
- Avoid clichés: brain/chip/robot; avoid 3D/gloss/heavy shadows

## Typography
- Font family (preferred): **Inter / Segoe UI / SF Pro** (neutral sans‑serif)
- Wordmark: **bold / semibold** (700–600)
- Secondary text: regular/medium (400–500)

## Colors
- Background gradient (recommended): **#061529 → #032E37**
- Wordmark on dark backgrounds: **#F5F7FA**
- Wordmark on light backgrounds (theme variant): **#061529**
- Mark layers (top → bottom):
  - Top: **#2DD4BF**
  - Middle: **#3B82F6**
  - Bottom: **#22C55E**

## Mark meaning
- 3 layers → 3‑layer architecture (**core rules / knowledge base / skills & workflows**)
- Foundation + growth: widths expand downward; stack reads as steps/base

## Mark geometry (sandwich)
- 3 rounded rectangles stacked vertically, **right edges aligned**
- Width ratios (bottom = 1.00): **top 0.584 / mid 0.788 / bottom 1.000**
- Gap to layer ratio: **gap = 0.36 × layer height**
- Bottom width vs stack height: **0.76 × stack height**
- Corner radius: **~0.33 × layer height**

## Fixed sizing rule
- **The mark (“sandwich”) height is ALWAYS equal to the cap‑height of the capital “A” in the wordmark “Agentify”.**
  - Applies to any lockup: README, cards, covers, social previews.

## Lockup layout
- Mark on the left, wordmark on the right
- Mark → word spacing: ~**0.22×(mark height)** (optically balanced)
- Clear space around the lockup: ≥ **1×(layer height)**

## Alignment axis for secondary lines
- The **left edge of the mark** is the base alignment axis for secondary lines
  - tagline / subtitle
  - supporting copy under the lockup

## Background styling (optional)
- Subtle grid is allowed as a structural texture
- Keep opacity very low (≈ 5–6%)
- Avoid crossing dense grid lines through key text in tight layouts

## Theme assets (recommended)
- Provide **two** logo variants:
  - Dark theme: white wordmark (**#F5F7FA**)
  - Light theme: navy wordmark (**#061529**)
