# Agentify Documentation

Agentify is a tool-agnostic framework for AI coding agents. It helps teams bootstrap safe agent behavior with an `AGENTS.md` foundation, then grow with project knowledge base files, skills, and workflows.

## Start Here

- [Getting Started](https://1kurepin.github.io/agentify/getting-started/)
- [Customization Guide](https://1kurepin.github.io/agentify/customization/)
- [Design Philosophy](https://1kurepin.github.io/agentify/philosophy/)
- [FAQ](https://1kurepin.github.io/agentify/faq/)

## What Agentify Covers

- AI coding agent rules and safeguards
- `AGENTS.md` templates for project setup
- Layered project knowledge base in `docs/`
- Reusable skills and workflow prompts
