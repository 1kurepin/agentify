# KB Draft Templates

Templates used by `.agentify/bootstrap-prompts/repo-scan.prompt.md` to generate draft KB files in `temp/kb-drafts/`.

## Templates

- `project_context.draft.template.md`
- `domain.draft.template.md`
- `glossary.draft.template.md`
- `architecture.draft.template.md`
- `constraints.draft.template.md`

## Usage

1. Copy each template to `temp/kb-drafts/`
2. Remove `.template` from filename
3. Fill with repository evidence
4. Keep unknowns as `TBD`
